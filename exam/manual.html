1. はじめに（必ず確認） ▲ 上へ

  • Google Chrome で受験してください（Edge・Safari・Firefox等は不可）。
  • 受験中は全画面（最前面）になります。試験終了まで全画面のままです。
  • 他のアプリ・タブへ切り替える（背面にする）と強制終了します。
  • 全画面解除・画面分割（Split View / スナップ）・ウィンドウ縮小・外部ディスプレイ接続・再読み込み・ページ移動も強制終了の対象です。
  • 外部ディスプレイ・プロジェクタは開始前に取り外してください（接続中は開始できません）。
  • Siri等の音声アシスタントやスマートフォン等の別端末の使用も不正行為です。
  • 強制終了が 3回 に達すると、不正とみなされ受験できなくなります。
  • 解答は教員のサーバーへ提出され、自分の端末には残りません。
  • 試験には開始・終了時刻があります。開始前は自動でカウントダウン待機し、終了時刻になるとその時点の解答が自動的に提出されます。
重要：この試験は Windows または macOS のデスクトップ版 Google Chrome でのみ動作します。スマートフォン・タブレット・Chromebook 等の端末では受験できません。必ずデスクトップPCで受験してください。
受験中にやってはいけないこと：別アプリ・別タブ・別ウィンドウへの切り替え、Esc等での全画面解除、画面の最小化、画面分割・ウィンドウ縮小・外部ディスプレイの接続・ページの再読み込み/移動/閉じる。いずれも「強制終了」としてカウントされ、サーバーにも記録されます。なお、受験中は右クリック・問題文のコピー・外部からの貼り付け・印刷はできません。

2. 受験の流れ ▲ 上へ

  1. 配布された受験開始ページのURLを Chrome で開く
  2. 大学Googleアカウント（soka-u.jp）でログインし、「試験を開始する」を押す
    ※ログイン画面はGoogleの公式画面です。表示されたアカウント候補から自分の大学アカウントを選択してください。
  3. 試験ページ（別ウィンドウ）が開く。注意事項を読み 「全画面で試験を開始する」を押す → 全画面になる
  4. 各問に解答する（記述・図形・プログラム・選択式）
  5. すべて終わったら右上の 「提出して終了」 を押す
  6. 確認画面で 「はい（提出する）」 を押すと、解答が提出されて終了
ポップアップ（試験ページ）がブロックされたら、許可してから再度「試験を開始する」を押してください。

4. 図形描画ツールの操作 ▲ 上へ

ユースケース図・E-R図・クラス図・PERT図・シーケンス図などのUMLや、V字モデル等を作図できます。

ボタン機能
↖ 選択/移動図形をクリックで選択、ドラッグで移動。選択中にDeleteで削除
▭ 四角ドラッグで四角形（□）
◯ 円ドラッグで楕円／円（◯）
／ 直線ドラッグで直線
→ 矢印ドラッグで矢印付き直線
A 文字クリックした位置に文字入力。文字をダブルクリックで再編集
● 黒 / ● 赤線の色を選択（塗りは常に透明）
コピー / 貼り付け選択中の図形をコピー（⌘/Ctrl+C）して貼り付け（⌘/Ctrl+V）。⌘/Ctrl+Dでその場で複製。同じ図形を何度も描かずに済みます
削除 / 元に戻す / 全消去選択図形の削除 / 取り消し / すべて消去
作図のコツ：四角や円を描いてから「A 文字」で枠内に名前を置くと、クラス図やPERTノードが作れます。関連・遷移は「→ 矢印」で表します。同じ大きさのノードを並べたいときは、1つ作ってコピー＆ペースト（複製）すると速くきれいに揃います。

ここで実際に試してみましょう（練習用）

下のエディタは本番とまったく同じ図形描画ツールです。自由に図形を描いて、移動・変形・コピー＆ペースト・削除・元に戻すなどを練習してください。操作に慣れておくと本番で戸惑いません。

※ ここでの練習内容は保存・採点・提出されません。何度でも自由に試せます。

うまくいかないときは「元に戻す」で一手ずつ戻せます。最初からやり直すときは「全消去」を押してください。

5. 提出のしかた ▲ 上へ

  1. 画面右上の「提出して終了」を押す
  2. 確認画面（ページ内に表示）で「はい（提出する）」を押す
    ※確認画面の表示中も背面化は検知されます。確認中に他アプリへ切り替えないでください。
  3. 「提出が完了しました」と表示されたら終了です。ウィンドウは閉じてかまいません
解答は教員のサーバーへ送信されます。自分の端末には解答・問題は残りません。
提出後に「サーバー送信に失敗しました」と出た場合は、すぐに担当教員へ申し出てください。

6. 困ったとき（よくある質問） ▲ 上へ

症状対処
「Chromeで受験してください」と出るGoogle Chrome で開き直す。Chromeでも出る場合は、プライバシー系ブラウザ(Brave等)や拡張機能を無効化
試験ページ（別ウィンドウ）が開かないポップアップブロックを許可してから、再度「試験を開始する」
大学Googleアカウントでログインできないsoka-u.jpアカウントでログインしているか確認。個人のGoogleアカウントでは受験できません
全画面にならない「全画面で試験を開始する」ボタンを必ずクリック（ボタン操作が全画面の条件）
「受験できません」と表示された強制終了が3回に達しています。担当教員に申し出てください
誤って提出してしまった担当教員に申し出てください
